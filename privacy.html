Kebijakan Privasi AI RAW

Terakhir diperbarui: 2 Januari 2026

Selamat datang di AI RAW Unlimited. Kami sangat menghargai privasi Anda sebagai pengguna. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan platform chat kami.

1. Informasi yang Kami Kumpulkan

Saat Anda menggunakan AI RAW, kami mengumpulkan beberapa jenis informasi melalui teknologi yang terintegrasi di sistem kami:

  • Data Autentikasi Google: Jika Anda masuk menggunakan fitur Google Sign-In, kami menerima data publik Anda seperti Nama, Alamat Email, dan Foto Profil untuk personalisasi pengalaman chat Anda.
  • Data Chat & Gambar: Kami memproses teks pesan dan gambar yang Anda unggah untuk memberikan respons AI. Data ini dikirimkan ke server API kami secara aman.
  • Data Teknis: Kami menggunakan Local Storage pada browser Anda untuk menyimpan riwayat percakapan (Chat History), preferensi tema (Dark/Light), dan pengaturan persona secara lokal di perangkat Anda.

2. Penggunaan Data

Informasi yang dikumpulkan digunakan semata-mata untuk:

  • Menyediakan layanan asisten AI yang responsif (Streaming AI).
  • Memungkinkan Anda untuk melanjutkan percakapan sebelumnya melalui fitur Riwayat Chat.
  • Menampilkan identitas pengguna pada antarmuka aplikasi.

3. Penyimpanan Data Lokal

Penting untuk diketahui bahwa riwayat percakapan Anda disimpan di Local Storage browser Anda. Artinya, data chat tersebut berada di perangkat Anda sendiri dan akan terhapus jika Anda membersihkan cache browser atau menggunakan tombol "Hapus Riwayat" di sidebar aplikasi kami.

4. Keamanan Data

Kami menggunakan enkripsi protokol standar untuk pengiriman data antara browser Anda dan server kami. Namun, perlu diingat bahwa tidak ada metode transmisi data di internet yang 100% aman.

5. Perubahan Kebijakan

Kami berhak memperbarui kebijakan ini sewaktu-waktu. Perubahan akan segera berlaku setelah dipublikasikan di halaman ini. Kami menyarankan Anda untuk memeriksa halaman ini secara berkala.
